# TrueArt | AI Analysis System

A modern, minimalistic interface for verifying digital authenticity and art provenance using neural precision systems.

## Vision
To provide a clean, high-performance environment for distinguishing between human artistry and synthetic origin.

## Core Features
- **Neural Scan**: High-precision image analysis with real-time feedback.
- **Glassmorphism UI**: Minimalist components designed for focus and clarity.
- **Benchmarking**: Integrated sample grid for neural comparison.

## Tech Stack
- **Frontend**: Next.js 15, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: Python (Flask), Neural Network Analysis (Deep Learning).

## Getting Started

### Prerequisites
- Node.js 18+
- Python 3.9+ (for backend analysis)

### Installation
1. Clone the repository.
2. Install frontend dependencies:
   ```bash
   cd frontend
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Development
The frontend is built with a custom minimalistic design system configured in `src/app/globals.css`. Components leverage Framer Motion for smooth, hardware-accelerated animations.
